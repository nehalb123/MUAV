A project based on ROS and Artificial Intelligence

Insight

This UAV will help the first responders to get acquainted of the surroundings before-hand. This would help pre-planning and also fasten the process of search and rescue operations.
The first responder will be assisted by a robot which will enter the scenario initially. This robot will autonomously trace a path between obstacles and debris lying around the scene. While tracing the path, the robot will map the surrounding area using a depth mapping sensor/camera. This data will be fed back to a host system which will develop a 3D map to assist the officials in making decisions. Sensors on the robot will also convey vital information regarding environmental parameters surrounding the scenario. Locomotion is done with the help of Brush-less DC Motors interfaced with the processor through a motor driver. The Industrial Grade motor drivers are internally interfaced using the CAN protocol and controlled using the serial port of the processor. Commands to the motor driver are sent with the help of Bash files (in the form of direction and RPM) also crucial in decision making. Kinect produces a cloud point matrix with the help of IR radiations. This data is then fed to the GPU. The Robotic Operating System (ROS) running on the processor interprets this data. It has a built-in library for Kinect and using this the 3D map of the data is processed. The cloud point matrix given by Kinect is sent through Wi-Fi to the host.
